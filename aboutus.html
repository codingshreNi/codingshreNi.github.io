ମୁଁ ଜଣେ Engineering ଛାତ୍ର | କେତେ ଦିନ ତଳେ ମୁଁ ଗୋଟିଏ Google assistant ପରି voice assistant ଟିଏ ବନେଇବା କଥା ଭାବିଥିଲି ହେଲେ କୌଣସୀ ପ୍ରକାରର voice assistant ବନେଇବାକୁ ଗୋଟିଏ ବହୁତ ବଡ dataset ଆବଶ୍ଯକ ଯେଉଁଥିରେ ବହୁତ ଗୁଡ଼ାଏ ଉଚ୍ଚାରଣ ଏବଂ ଲେଖା ଶୈଳୀ ସମ୍ବନ୍ଧିତ ସୂଚନାସୂଚନା ଥାଏ। ସେମିତି dataset online ମିଳିବା ଅତି କଷ୍ଟ,ଏବଂ ତାହାର କାରଣ ହେଉଛି ଓଡ଼ିଆ ବିଦ୍ୟାଳୟ ଗୁଡିକରେ computer ଶିକ୍ଷ୍ୟା ପ୍ରତି ଅବହେଳା। ଏହି website ମାଧ୍ୟମରେ ମୁଁ ସେହି ସମସ୍ୟା ର ଗୋଟିଏ ଛୋଟ ସମାଧାନଟିଏ ଯୋଗଦେବାକୁ ଚେଷ୍ଟା କରୁଛି।

Python ହେଉଛି ବହୁ computer ଭାଷା ମଧ୍ୟରେ ଗୋଟିଏ। ଏହା ଶିଖିବା ସହଜ ଏବଂ ଏହାର ଉପଯୋଗିତା ବହୁତ। ଆଜିକାଲିର tech ଜଗତରେ python ଶିଖିଥିବା programmer ମାନଙ୍କ ବହୁତ ଚାହିଦା ରହିଛି। ଆମେ ଏହି ୱେବସାଇଟ ରେ programming ଶିଖିବା , ଅର୍ଥାତ୍ ଆମେ ଶିଖିବା program କେମିତି ବନଯାଏ।

ଏହି website ରେ ଥିବା tutorial ଗୁଡାକ ଆରମ୍ଭ କରିବା ଆଗରୁ କେତେକ minimum ଜ୍ଞାନ ଆବଶ୍ଯକ :

  • computer ଚଲେଇବା କିମ୍ବା operate କରିବାର ଧାରଣା ଥିବା ଆବଶ୍ଯକ |
  • କୌଣସୀ program ବା software install/load କରିପାରୁଥିବା ଦରକାର
  • minimum english ଜ୍ଞାନ ଥିବା ଜରୁରୀ କାରଣ କେତେଗୁଡିଏ ଶବ୍ଦ ବ term ର ଓଡିଆ ରେ translation ବୁଝିବା କଶ୍ଟ |
